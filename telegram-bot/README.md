# Бот-расшифровка анализов и заключений врачей

Бот помогает разобрать медицинские документы (анализы, выписки, заключения) простым языком — удобно для пожилых родственников. Пришли фото или снимок экрана — бот объяснит, что в норме, на что обратить внимание и что делать дальше. Можно также задавать вопросы текстом.

## Как запустить

1. **Telegram-бот:** создай бота в [@BotFather](https://t.me/BotFather) (`/newbot`), скопируй токен.

2. **Нейросеть (достаточно одной):**
   - **Groq (бесплатный тариф)** — [console.groq.com](https://console.groq.com), создай API-ключ.
   - **OpenAI (платно после бесплатного кредита)** — [platform.openai.com/api-keys](https://platform.openai.com/api-keys). Используются GPT-4o (фото) и GPT-4o-mini (текст).

   Если в `.env` указаны оба ключа, бот сначала использует Groq.

3. Создай файл `.env` в папке проекта:
   ```bash
   cp .env.example .env
   ```
   Заполни в `.env`:
   - `BOT_TOKEN` — токен от BotFather
   - `GROQ_API_KEY` и/или `OPENAI_API_KEY` — хотя бы один из них

4. Установи зависимости и запусти (обязательно используй виртуальное окружение или установи пакеты в тот же Python, которым запускаешь):
   ```bash
   python3 -m venv .venv
   .venv/bin/pip install -r requirements.txt
   .venv/bin/python bot.py
   ```
   Если окружение уже создано:
   ```bash
   .venv/bin/python bot.py
   ```
   Запуск просто `python bot.py` без активации venv приведёт к ошибке, если пакеты не установлены в системный Python.

## Что умеет бот

- **Фото анализов/заключений** — пришли снимок или фото документа; бот прочитает и объяснит простыми словами: что в норме, что не так, что делать дальше.
- **Файл-картинка** — можно отправить изображение как документ (JPEG/PNG). PDF пока не поддерживается — лучше отправить скриншот страницы.
- **Вопрос текстом** — например: «что значит повышенный креатинин?» — бот ответит так же простым языком.

## Команды

- `/start` — приветствие и краткая инструкция
- `/help` — справка по командам и возможностям

## Важно

Бот даёт только пояснения и не заменяет врача. При серьёзных отклонениях или сомнениях нужно обращаться к доктору.
